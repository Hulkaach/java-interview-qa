### Структура классического проекта
Чтобы код был логически структурирован, в веб-приложениях принято выделять как минимум 
три группы классов (их больше, но как минимум эти всегда можно использовать), 
для которых предоставляют отдельные директории.

Перечислим их:

- Model.  
- Controllers.  
- Service.
